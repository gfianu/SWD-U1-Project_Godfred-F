# ChemLearn

## Overview

**ChemLearn** is an educational web application designed to help students understand organic chemistry concepts through structured content, interactive quizzes, and a clean, user-friendly interface.

This project reflects my background as a college chemistry professor learning software development, combining educational expertise with modern front-end technologies to build intuitive learning tools.

---

## Purpose

ChemLearn was built to address common challenges in chemistry education by:

- Organizing complex material into clear, navigable sections  
- Supporting active learning through quizzes and discussion  
- Demonstrating how thoughtful UI design can improve comprehension  

The application also serves as a portfolio project showcasing my skills in **JavaScript, React, and user-centered design**.

---

## Features

- **Lecture Library**
  - Topic-based organization
  - Dedicated pages for videos, notes, and quizzes

- **Interactive Quizzes**
  - Multiple-choice questions tied to lecture content
  - Score calculation and feedback

- **Dashboard**
  - Overview of lectures and quizzes
  - Clean, card-based layout for usability

- **Comments & Discussion**
  - Allows learners to engage with content
  - Encourages reflection and interaction

- **Responsive Design**
  - Optimized for desktop, tablet, and mobile devices

## Design & User Experience

ChemLearn prioritizes:

- Clear navigation and logical information flow  
- Minimal visual clutter to reduce cognitive load  
- Components designed from a learner-first perspective  

Design decisions were informed by real classroom teaching experience and student feedback.
